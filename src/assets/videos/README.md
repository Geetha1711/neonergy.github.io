# Hero Video

Drop `hero.mp4` in this directory.

Specs:
- Resolution: 1920×1080
- Format: H.264 MP4
- Max size: 8 MB
- Duration: 10–20 seconds, seamlessly looping
- Audio: silent (muted on the page, override not possible)
- Content: subtle motion — no rapid cuts

Also add `hero-poster.jpg` to `src/assets/images/` as a fallback.
